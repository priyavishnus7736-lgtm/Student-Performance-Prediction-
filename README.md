# Student Performance Analysis & Prediction System

An end-to-end Data Science framework designed to evaluate structural academic parameters and forecast student final performance tracking records using machine learning regression models.

## 📊 Dataset Dictionary
The pipeline models student behaviors spanning 11 fundamental system features:
* **Target Feature:** `Final Score` (Numerical evaluation)
* **Behavior Profiles:** `Gender`, `Study Hours`, `Attendance (%)`, `Previous Marks (%)`, `Participation`, `Assignments Score`, `Projects Score`, `Test Score`, `Behavior`, and `Extracurricular Activities`.

## ⚙️ Methodology & Architecture
1. **Data Preprocessing:** Outlier handling and categorical Label Encoding transforms.
2. **Feature Engineering:** Extracts *Total Assessment Scores*, *Study Efficiency Coefficients*, and *Attendance-to-Study Ratios*.
3. **Model Evaluation:** Benchmarks error matrices across Linear Regression, Decision Trees, Random Forests, and Support Vector Regression (SVR).
